# Contributing Guidelines

Welcome to the ASSAI Assessment Prep project! We appreciate your interest in contributing. Here are guidelines to help you get started:

## How to Contribute
1. **Fork the Repository**: Start by forking the repository to your own GitHub account.
2. **Clone Your Fork**: Clone the forked repository to your local machine.
3. **Create a Branch**: Create a new branch for your feature or bug fix. Use a descriptive name that explains the changes you are making.
4. **Make Changes**: Implement your changes in your local repository. Make sure to include appropriate comments and documentation.
5. **Commit Changes**: Commit your changes with a concise, meaningful commit message.
6. **Push to Your Fork**: Push your changes back to your fork on GitHub.
7. **Create a Pull Request**: Go to the original repository and create a pull request from your branch.

## Code of Conduct
Please adhere to the [Code of Conduct](CODE_OF_CONDUCT.md) while contributing.

## Reporting Issues
If you encounter bugs or issues, please open an issue in the repository. Include relevant details to help us understand the problem more clearly.

Thank you for your contributions!
